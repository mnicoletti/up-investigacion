# Proyecto de Investigación y Desarrollo

## Descripción

Este proyecto tiene como objetivo la creación de un paper de investigación sobre un tema a elección. Debe ser aprobado por el profesor a cargo. El paper debe ser escrito en LaTeX y debe contener una revisión bibliográfica, una propuesta de solución y una evaluación de la propuesta.
